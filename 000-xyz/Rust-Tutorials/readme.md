Rust 输出文字的方式主要有两种：`println!()` 和 `print!()`

```rs

fn main () {
  // print
  print!("print 🦀");
}

/*

$ rustc ./print.rs && ./print

*/


```

```rs
fn main () {
  // println! === print! + \n
  println!("println! === print! + \\n 🦀");
}


/*

$ rustc ./println.rs && ./println

*/
```
